# Media Library

## 1. 프로젝트 개요
도서(Book)와 영화(Movie) 컬렉션을 효율적으로 관리하기 위해 객체지향 원칙을 적용하여 설계된 파이썬 패키지입니다. 

## 2. 설치 방법
이 패키지는 로컬 환경에서 설치하여 사용할 수 있습니다. 터미널에 아래 명령어를 입력하세요.

git clone [https://github.com/Joono1229/media_library.git](https://github.com/Joono1229/media_library.git)
