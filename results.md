# 기말 프로젝트 과제 실행 결과 요약

## 1. pycodestyle 스타일 검사 결과
`pycodestyle media_library/` 실행 시 경고 및 에러가 단 1건도 발생하지 않고 완벽하게 통과 완료. (PEP 8 준수 완료)
"pytest" 실행 시 모든 테스트 통과 완료.

```bash
$ pycodestyle media_library/
# [출력 결과 없음 - 경고 0건으로 완벽 통과]
$ pytest
# [============================================ 9 passed in 0.07s ============================================]
#총 9개의 테스트 케이스를 작성하였고 pytest 실행시 모두 성곡적으로 통과함.
